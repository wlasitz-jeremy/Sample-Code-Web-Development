Notes on Forms

Form Field Specific Notes

  • GET Request method requests information from the server
  • POST Request method sends information to the server
  • Action attribute is the url that the user is taken to on submit
  • label for attribute connects the label to the element with the matching id
  • the name attribute in an input field labels the information to send it to the server
  • required field will stop the form from submitting without it being entered
  • minlength and maxlength can be used to specify field length to the user
  • checkboxes are designed for users to be able to select more than 1 option
  • radio buttons are used when a user can only select 1 thing from a set
  • in radio buttons, the value attribute is the information that is sent to the server. it's referred to as whatever is specified in the name attribute.

Notes not specific to forms

  • class is used to assign css or javascript to that particular element
  • Divs are used for grouping information. helps when elements are inline but need to be in blocks
